<p style='font-size:25px;'>🚩 升旗快报生成器</p><br>
<br>
一个简单易用的在线工具，用于快速生成学校升旗仪式的快报图。通过自定义关键信息和布局，你可以轻松制作出专业且美观的升旗仪式宣传图片。<br>
<br>
✨ 主要功能<br>
可视化编辑： 通过表单输入信息，实时预览生成的快报效果。<br>
信息自定义： 轻松修改标题（如升旗快报）、组织介绍、人员信息（主持人、护旗手、发言人等）。<br>
图片上传： 支持上传自定义人物图片并裁剪。<br>
背景样式： 可切换背景风格（如液态玻璃、模糊化等）。<br>
布局精调： 提供更多设置，允许用户自定义快报中部分文本块（标题、人员信息）的 X/Y 坐标，实现精确排版。<br>
本地保存： 一键生成图片并保存到本地。<br>
<br>
💻 如何使用<br>
访问项目网站： jhcwcolin.github.io<br>
填写基础信息： 在右侧的输入框中填入周期数（如第 X 周）、班级介绍和人员信息。<br>
上传背景图片： 点击“上传图片”选择你拍摄的人物图片。<br>
进入“更多设置”，拖动或输入坐标值来微调文字的位置，达到最佳布局。<br>
生成并保存： 右键图片区域即可下载生成的 PNG 图片。<br>
